# dawn
<h1>Dawn Theme Customization</h1>
  <p>
    This project is built on <strong>Shopify’s Dawn theme</strong>, extending its
    performance-focused foundation with <strong>custom-built sections</strong>
    and flexible layout controls.
  </p>

  <div class="section">
    <h2>Project Overview</h2>
    <p>
      The theme has been enhanced with multiple custom sections including
      an <strong>Announcement Bar</strong>, <strong>Header</strong>,
      <strong>Hero Section</strong>, and <strong>Collection-based layouts</strong>.
      All sections are modular, reusable, and fully configurable via the
      Shopify Theme Editor.
    </p>
  </div>

  <div class="section">
    <h2>Custom Sections</h2>
    <ul>
      <li>Announcement Bar with dynamic text and link support</li>
      <li>Custom Header aligned with Dawn’s structure</li>
      <li>Hero Section with image, text, and layout controls</li>
      <li>Featured Collections and Product-based sections</li>
      <li>Reusable content sections using blocks</li>
    </ul>
  </div>

  <div class="section">
    <h2>Key Features</h2>
    <ul>
      <li>Built with Shopify Liquid, JSON schema, HTML, CSS, and JavaScript</li>
      <li>Fully editable via Shopify Theme Editor</li>
      <li>Mobile-first and responsive design</li>
      <li>Performance-optimized and lightweight</li>
      <li>Clean section architecture following Dawn best practices</li>
    </ul>
  </div>

  <div class="section">
    <h2>Customization Highlights</h2>
    <p>
      Each section includes configurable settings such as text content,
      images, links, typography, spacing, and layout direction. This ensures
      merchants can customize the storefront without modifying code.
    </p>
  </div>

  <div class="section">
    <h2>Technology Stack</h2>
    <span class="badge">Shopify Liquid</span>
    <span class="badge">HTML</span>
    <span class="badge">CSS</span>
    <span class="badge">JavaScript</span>
    <span class="badge">JSON Schema</span>
  </div>

  <div class="section">
    <h2>Responsive & Performance</h2>
    <p>
      The theme maintains Dawn’s accessibility and performance standards while
      introducing custom layouts. All sections are tested across desktop,
      tablet, and mobile devices.
    </p>
  </div>
